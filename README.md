# Supermarket Simulator — Save Editor

Внешний редактор сохранений для игры **Supermarket Simulator** (Nokta Games).  
Не ставит моды в папку игры и ничего не инжектит — только читает и пишет `.es3` файлы.

## Возможности

- Редактирование денег (`Money` / `Funds`)
- Название магазина, день, кассы, полки, сотрудники, уровень
- Разблокировка лицензий (ID **21–47**)
- Автобэкап перед каждым сохранением
- Восстановление из последних 5 копий
- Редактор сырого JSON
- Предупреждение, если игра запущена (`psutil`)
- Полностью русский интерфейс

## Требования

- Windows 10/11 (целевая платформа)
- Python **3.11+** (если запускаете из исходников)
- Игра должна быть **закрыта** при записи сохранения

## Установка из исходников

```bat
git clone <url-репозитория>
cd Mod-Supermarket-Simulator-
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Сборка .exe

```bat
build.bat
```

Готовый файл: `dist\SupermarketSaveEditor.exe`

## Как пользоваться

1. Запустите редактор.
2. Если сохранение найдено автоматически — подтвердите загрузку.  
   Иначе нажмите **«Открыть сохранение»** и выберите `.es3`.
3. Стандартный путь Windows:
   ```
   %USERPROFILE%\AppData\LocalLow\Nokta Games\Supermarket Simulator\SaveFile.es3
   ```
4. Измените нужные поля на вкладках.
5. Нажмите **«Сохранить изменения»** (бэкап создаётся всегда).
6. Запустите игру и загрузите сохранение.

## Важно про формат

Сохранения — Easy Save 3 (`.es3`), обычно без шифрования, JSON с обёртками `__type` / `value`.

| Поле в UI | Ключ в файле |
|-----------|--------------|
| Деньги | `Progression.value.Money` |
| День | `Progression.value.CurrentDay` |
| Лицензии | `Progression.value.UnlockedLicenses` |
| Уровень | `Progression.value.CurrentStoreLevel` |

Лицензии начинаются с **ID 21**. ID ниже 21 могут сломать разблокировку в игре.
