# Supermarket Simulator — Редактор сохранений

Внешняя программа для изменения сохранений игры **Supermarket Simulator**.  
Это **не мод**: ничего не кладётся в папку игры и ничего не «вшивается» в процесс.  
Программа просто открывает файл сохранения (`.es3`), меняет нужные числа и сохраняет обратно.

---

## Главное правило (прочитайте первым!)

### Игра должна быть ЗАКРЫТА

| Когда | Игра |
|-------|------|
| Установка / первый запуск редактора | Можно закрыта, неважно |
| Редактирование и кнопка **«Сохранить изменения»** | **Обязательно закрыта** |
| После сохранения — проверка результата | **Запускаете игру** и загружаете сейв |

Если игра открыта и вы сохраните файл — изменения могут **не примениться** или сейв может **сломаться**.  
Редактор сам предупредит, если увидит запущенный `Supermarket Simulator`.

---

## Что умеет редактор

- Менять деньги
- Менять название магазина, день, кассы, полки, сотрудников, уровень
- Разблокировать лицензии товаров
- Делать резервную копию **перед каждым сохранением** (автоматически)
- Восстанавливать сейв из бэкапа
- Показывать «сырой» JSON для продвинутых

Интерфейс полностью на русском.

## Способ А — самый простой (двойной клик)

Подходит, если у вас уже скачана папка с проектом.

### Шаг 1. Установите Python (один раз)

1. Откройте сайт: https://www.python.org/downloads/
2. Скачайте **Python 3.11 или новее** (кнопка Download).
3. Запустите установщик.
4. **ОБЯЗАТЕЛЬНО** поставьте галочку внизу окна:
   - **Add python.exe to PATH**
5. Нажмите **Install Now** и дождитесь конца установки.
6. Если спросите про Disable path length limit — можно согласиться.

Проверка: откройте **Командную строку** (`Win + R` → `cmd` → Enter) и введите:

```bat
python --version
```

Должно показаться что-то вроде `Python 3.12.x`.  
Если пишет «не найдено» / `not recognized` — Python не в PATH, переустановите с галочкой выше.

### Шаг 2. Скачайте этот проект

- Через GitHub: кнопка **Code → Download ZIP** → распакуйте архив в удобную папку  
  (например `C:\Games\Mod-Supermarket-Simulator-`).
- Или клонируйте через git, если умеете.

### Шаг 3. Запустите редактор

1. Откройте папку проекта.
2. Найдите файл **`запуск.bat`** (или **`start.bat`** — то же самое).
3. **Дважды кликните** по нему.
4. При первом запуске подождите: скачаются нужные библиотеки (нужен интернет).
5. Откроется окно:  
   `Supermarket Simulator — Save Editor v1.0`

Готово. Консоль можно не трогать — `запуск.bat` всё делает сам.

## Способ Б — через консоль (если bat не сработал)

1. **Закройте игру.**
2. Откройте папку проекта в проводнике.
3. В адресной строке проводника напишите `cmd` и нажмите Enter  
   (откроется консоль уже в нужной папке).
4. По очереди введите команды (после каждой — Enter):

```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

После последней команды откроется окно редактора.

## Как пользоваться редактором (пошагово)

### 1. Закройте Supermarket Simulator полностью

Проверьте в диспетчере задач (`Ctrl + Shift + Esc`), что процесса игры нет.

### 2. Запустите редактор

Через `запуск.bat` или `python main.py`.

### 3. Откройте файл сохранения

При старте программа сама ищет сейв и может спросить: «Загрузить?» — нажмите **Да**.

Если не нашла — нажмите **«Открыть сохранение»** и укажите файл вручную.

Обычный путь на Windows:

```text
C:\Users\ВАШЕ_ИМЯ\AppData\LocalLow\Nokta Games\Supermarket Simulator\SaveFile.es3
```

Быстрый способ открыть папку:

1. Нажмите `Win + R`
2. Вставьте:

```text
%USERPROFILE%\AppData\LocalLow\Nokta Games\Supermarket Simulator
```

3. Enter  
Нужный файл обычно называется **`SaveFile.es3`**.

> Папка `AppData` скрыта — через `%USERPROFILE%\AppData\...` она всё равно откроется.

### 4. Измените нужное

- **💰 Деньги** — баланс, кнопки +10 000 / +100 000 и т.д.
- **🏪 Магазин** — название, день, кассы, полки, сотрудники
- **📦 Лицензии** — галочки или «Разблокировать все»
- **🔧 Прочее** — бэкапы и сырой JSON (осторожно)

### 5. Сохраните

Нажмите **«Сохранить изменения»** внизу окна.

- Редактор **сам** сделает резервную копию в папку `backups` рядом с сейвом.
- Должно появиться сообщение об успехе.

### 6. Запустите игру и проверьте

1. Откройте **Supermarket Simulator**.
2. Загрузите своё сохранение.
3. Проверьте деньги / лицензии / день.

Если что-то не так — во вкладке **Прочее** можно восстановить бэкап.

## Сборка в один .exe (по желанию)

Чтобы не ставить Python на другом ПК:

1. Убедитесь, что Python установлен (см. выше).
2. Дважды кликните **`build.bat`** (или запустите его из консоли).
3. Дождитесь конца сборки.
4. Готовый файл:

```text
dist\SupermarketSaveEditor.exe
```

Его можно копировать и запускать двойным кликом.  
Игру при сохранении по-прежнему нужно **закрывать**.

## Если что-то пошло не так

| Проблема | Что делать |
|----------|------------|
| `python` не найден | Переустановите Python с галочкой **Add to PATH** |
| Редактор не находит сейв | Откройте файл вручную через **«Открыть сохранение»** |
| «Игра запущена» | Полностью закройте игру и сохраните снова |
| После правок в игре старые данные | Значит сохранили при открытой игре, или загрузили не тот слот — восстановите бэкап |
| Сейв повреждён | Вкладка **Прочее** → восстановить из резервной копии |
| Хочу потренироваться без риска | Откройте тестовый файл `samples\SaveFile.es3` |

Лог ошибок пишется в файл **`editor.log`** рядом с программой.

---

## Важно про лицензии

В игре лицензии имеют номера **с 21 по 47**.  
Номера меньше 21 лучше **не ставить** — могут сломать разблокировку через компьютер в игре.

## Для любопытных (не обязательно)

Структура сейва — Easy Save 3 (`.es3`), обычно обычный текст/JSON.

| В редакторе | В файле |
|-------------|---------|
| Деньги | `Progression` → `Money` |
| День | `CurrentDay` |
| Лицензии | `UnlockedLicenses` |

Тесты для разработчиков:

```bat
python -m unittest tests.test_save_handler -v
```
